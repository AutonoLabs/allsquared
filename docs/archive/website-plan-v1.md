# AllSquared Website Structure Plan

## Website Goals
- Showcase AllSquared as the UK's first integrated platform for secure service contracts
- Generate leads from freelancers, consultants, and service providers
- Build trust through clear value proposition and regulatory compliance messaging
- Drive sign-ups for early access/waitlist

## Target Audience
1. **Primary:** Freelancers, consultants, and self-employed professionals
2. **Secondary:** Home improvement clients, event service providers
3. **Tertiary:** Investors and partners

## Website Structure

### 1. Homepage
- Hero section with clear value proposition
- Problem statement (payment disputes, contract complexity)
- Solution overview (AI contracts + FCA escrow + milestones)
- Key features showcase
- Social proof / statistics
- CTA: Join waitlist / Request demo

### 2. How It Works
- Step-by-step workflow visualization
- 6-step process: Generate → Sign → Secure → Deliver → Release → Resolve
- Visual demonstrations
- CTA: Get started

### 3. Features
- AI Contract Generation (5 service categories)
- FCA-Backed Escrow
- Milestone Management
- LITL (Lawyer-in-the-Loop) optional service
- Dispute Resolution
- Each feature with icon, description, benefits

### 4. Pricing
- Free tier / Basic tier
- Professional tier
- Enterprise tier
- LITL add-on pricing
- Transparent fee structure
- CTA: Start free trial

### 5. For Freelancers
- Targeted landing page for service providers
- Pain points addressed
- Benefits specific to freelancers
- Success stories / use cases
- CTA: Join as provider

### 6. For Clients
- Targeted landing page for service buyers
- How to hire with confidence
- Payment protection benefits
- Quality assurance
- CTA: Post a project

### 7. About
- Mission and vision
- Team (placeholder with roles)
- Company values
- Regulatory compliance (SRA unreserved services)
- FCA escrow partnerships

### 8. Blog/Resources (optional for MVP)
- Legal guides
- Contract templates
- Industry insights

### 9. Contact
- Contact form
- Email: hello@allsquared.uk
- Social media links
- Office location (if applicable)

### 10. Legal Pages
- Terms of Service
- Privacy Policy
- Cookie Policy
- Acceptable Use Policy

## Design Principles
- Clean, modern, professional
- Trust-building elements (FCA logos, security badges)
- Mobile-first responsive design
- Fast loading times
- Accessibility compliant
- Brand colors: Deep blue (#1E3A8A), Bright green (#10B981), Orange (#F97316)
- Typography: Poppins font family

## Key Messaging
- **Headline:** "Secure Service Contracts for the UK's Freelance Economy"
- **Tagline:** "The only platform combining AI contracts, escrow, and milestone management"
- **Value Props:**
  - Generate professional contracts in minutes
  - Protect your payments with FCA-backed escrow
  - Track milestones and release payments automatically
  - Resolve disputes with AI-assisted mediation

## CTAs Throughout
- "Join Waitlist" (primary)
- "Request Demo"
- "Get Started Free"
- "Learn More"
- "Contact Sales"

## Technical Requirements
- Static site generation for performance
- SEO optimized
- Analytics integration ready
- Form submission handling
- Email capture for waitlist
- Social sharing meta tags

## Content Tone
- Professional yet approachable
- Clear and jargon-free
- Benefit-focused
- Trust-building
- Action-oriented
